    Read single or multiple values from a modbus slave depending on whether a Fixnum or a Range was given. Note that in the case of multiples, a pluralized version of the method is sent to the slave

    # File /Users/ned/.rvm/gems/ruby-1.9.2-p290/gems/rmodbus-1.1.0/lib/rmodbus/proxy.rb, line 24
def [](key)
  if key.instance_of?(Fixnum)
    @slave.send("read_#{@type}", key, 1)
  elsif key.instance_of?(Range)
    @slave.send("read_#{@type}s", key.first, key.count)
  else
    raise ProxyException, "Invalid argument, must be integer or range. Was #{key.class}"
  end
end
